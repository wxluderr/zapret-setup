> **Умный, быстрый и полностью автоматический установщик для [Zapret](https://github.com/Flowseal/zapret-discord-youtube).** > Забудьте про ручную распаковку архивов, возню с правами и правку конфигурационных файлов. Один запуск — и скрипт сделает всё за вас.

---

## ✨ Главные фичи

*  **Автоматическая установка:** Скрипт сам подключается к API GitHub, находит **самый свежий** релиз и скачивает его. Вам не нужно следить за версиями.
*  **Проверка прав:** Нет прав администратора? Не проблема. Скрипт сам вежливо запросит повышение привилегий и перезапустится. 
*  **Авто-конфиг:** Сам прописывает необходимые параметры: включает `game_filter` в режим `all` и активирует автопроверку обновлений.
*  **Авто-обновление hosts:** Подтягивает актуальный файл обхода блокировок `hosts` прямо из сети и бережно делает бэкап вашего оригинального системного файла.
*  **Очистка:** Убирает за собой все временные архивы и папки после завершения установки.

---

## 🚀 Установка и запуск

Всего два простых шага:

1. Скачайте файл `zapret_setup.bat` из этого репозитория.
2. Запустите его от имени Администратора (если запустили просто дабл-кликом, скрипт сам запросит права Администратора).
3. Зачилльтесь, скрипт всё сделает сам.

> **Примечание:** По завершении базовой подготовки скрипт автоматически откроет `service.bat` для финальной настройки службы.

---

## 🛠️ Как это работает под капотом

Инструмент написан на чистом `Batch`, но использует `PowerShell` для:
* Безопасного подключения к GitHub (с принудительным `TLS 1.2`).
* Парсинга JSON-ответов API для извлечения прямых ссылок на загрузку.
* Нативной распаковки `.zip` архивов без зависимости от сторонних программ (типа WinRAR или 7-Zip).
