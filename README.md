> **Умный, быстрый и полностью автоматический установщик для [Zapret](https://github.com/Flowseal/zapret-discord-youtube).** > Забудьте про ручную распаковку архивов, возню с правами и правку конфигурационных файлов. Один запуск — и скрипт сделает всё за вас.

---

## ✨ Главные фичи

* 🚀 **Smart-Download:** Скрипт сам подключается к API GitHub, находит **самый свежий** релиз и скачивает его. Вам не нужно следить за версиями.
* 🛡️ **Auto-Admin Check:** Нет прав администратора? Не проблема. Скрипт сам вежливо запросит повышение привилегий и перезапустится.
* 🖥️ **Full-Screen UI:** Автоматически разворачивает консоль на весь экран, стилизует текст и делает процесс установки наглядным (стиль "Матрицы" прилагается 🟢).
* ⚙️ **Auto-Config:** Сам прописывает необходимые параметры: включает `game_filter` в режим `all` и активирует автопроверку обновлений.
* 🌐 **Hosts Manager:** Подтягивает актуальный файл обхода блокировок `hosts` прямо из сети и бережно делает бэкап вашего оригинального системного файла.
* 🧹 **Zero-Trash:** Убирает за собой все временные архивы и папки после завершения установки.

---

## 🚀 Установка и запуск

Всего два простых шага:

1. Скачайте файл `zapret_setup.bat` из этого репозитория.
2. Запустите его двойным кликом (скрипт сам запросит права Администратора).
3. Откиньтесь на спинку кресла и смотрите, как магия происходит сама собой. ✨

> **Примечание:** По завершении базовой подготовки скрипт автоматически вызовет `service.bat` для финальной настройки службы.

---

## 🛠️ Как это работает под капотом

Инструмент написан на чистом `Batch`, но использует мощь `PowerShell` для:
* Безопасного подключения к GitHub (с принудительным `TLS 1.2`).
* Парсинга JSON-ответов API для извлечения прямых ссылок на загрузку.
* Нативной распаковки `.zip` архивов без зависимости от сторонних программ (типа WinRAR или 7-Zip).
